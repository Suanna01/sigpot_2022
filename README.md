# 2022 여기톤 "식팟"팀

---

### 대학생에게 가장 중요한 것은 바로 밥!!  <br>
### 공강 시간에 밥은 먹어야 하는데 밥 먹을 친구가 없을 때 이용하는 웹 사이트

---

### 프로젝트 개별 환경
- 개발환경 : Visual Studio code (VS code)
- 기술스택 : Django, HTML, CSS, JavaScript
- 형상관리 : Notion, GitHub

### 팀원
- 기획/디자인 : 곽은진(이화여대)
- 프론트엔드 : 장윤경(동덕여대), 허은(덕성여대)
- 벡엔드 : 김지수(숙명여대), 양성원(성신여대), 이하진(서울여대)
